# 毕业设计介绍
## 股票APP
### 主要功能：
1.用户收藏自己喜欢的股票，并对股票进行分组管理。
2.简单的模拟交易。
3.阅读最新股票资讯。
4.登录注册与个人信息、APP信息、APP更新等基础功能。

* 从右上角可打开股票搜索页面，或者更新整个自选页面。
* 从股票列表下拉更手动更新股票数据。
